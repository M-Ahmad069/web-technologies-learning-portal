 Web Technologies Learning Portal

A tutorial-style website built for my Web Technologies course assignment. The layout and style are inspired by W3Schools tutorial pages.

 Overview

This project is a simple learning site with 8 lecture pages that cover basic web tech topics—HTML, CSS, and how they fit together. Each lecture has explanations, code examples, and a way to try the code in your browser.

 Features

- Sidebar navigation – List of all lectures so you can jump to any topic
- Tutorial-style layout – Clear sections with headings and short explanations
- Code blocks – Examples you can copy, run, or reset
- Try it Yourself – Opens a small editor so you can run and edit the code
- Next / Previous – Move between lectures without going back to the home page
- Responsive layout – Works on different screen sizes

 Technologies Used

- HTML – Structure and content of all pages
- CSS – Styling, layout, and responsive behaviour
- JavaScript – Navbar toggle, code copy/run/reset, sidebar highlight

 Lectures

1. Introduction to Web Platforms  
2. Categories of Web Applications  
3. HTML Basics  
4. Tables, Div, Span and Layout  
5. HTML5 Introduction  
6. HTML Forms  
7. CSS Introduction  
8. CSS Selectors

 How to Run Locally

1. Clone or download the project.
2. Open `index.html` in your browser (double-click or use “Open with” and choose a browser).

You can also use a local server (e.g. Live Server in VS Code) if you prefer—just serve the folder that contains `index.html` and open it in the browser.
