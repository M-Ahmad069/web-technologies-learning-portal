CSS Selectors

Lecture 8 • Reading time: 8 minutes

CSS selectors are patterns used to target HTML elements and apply styles to them. This lecture covers element, descendant, child, and attribute selectors, plus pseudo-classes and pseudo-elements.

What Are CSS Selectors?

CSS selectors are patterns used to select HTML elements. They let you apply styles to specific elements—by tag name, relationship, attribute, or state. 

p {
  color: blue;
}

This selector applies the style to all <p> (paragraph) elements on the page. Selectors are the left-hand side of a CSS rule; the declaration block (inside the braces) is what gets applied to the matched elements.

Try it: Element Selector Example

Element selector styles all p tags. Run it to see blue text.

Example
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Selector Example</title>
<style>p { color: blue; }</style>
</head>
<body>
<p>This paragraph is styled with CSS.</p>
</body>
</html>

Descendant Selector

Code explanation: The descendant selector ol li selects all li inside ol at any nesting level. Run the example to see all list items styled blue.

Example
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Descendant Selector</title>
<style>ol li { color: blue; }</style>
</head>
<body>
<ol>
<li>List item one</li>
<li>List item two</li>
<ul><li>Sublist item A</li><li>Sublist item B</li></ul>
<li>List item three</li>
</ol>
</body></html>

This selector targets all li elements inside an ol—including those inside the nested ul. So “List item one”, “List item two”, “Sublist item A”, “Sublist item B”, and “List item three” would all be blue.

Child Selector

The child selector selects only direct child elements. It uses the > combinator between parent and child. 

ol > li {
  color: blue;
}

Difference from descendant: ol li matches every li inside ol (at any depth). ol > li matches only li elements whose direct parent is ol. So with the HTML above, “Sublist item A” and “Sublist item B” would not be selected by ol > li, because their direct parent is ul, not ol.

Attribute Selector

Attribute selectors select elements based on their attributes and optionally their values. They are written in square brackets.

Example HTML:

<ul>

<li><a href="https://google.com">Google</a></li>
<li><a href="https://facebook.com">Facebook</a></li>
<li><a href="https://amazon.com">Amazon</a></li>

</ul>

Example CSS:

a {
  color: green;
}

a[href="https://facebook.com"] {
  color: blue;
}

All links are green. The rule a[href="https://facebook.com"] targets only the anchor whose href is exactly https://facebook.com, so that link is blue. You can also use patterns like [href^="https"] (starts with) or [href*="google"] (contains).

Pseudo-Class Selectors

Code explanation: Pseudo-classes like :hover style elements in special states. This example changes the button background to orange when you hover. Run the code and hover over the button to see the effect.

Example
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Pseudo-Class Example</title>
<style>button:hover { background-color: orange; }</style>
</head>
<body>
<button type="button">Click Me!</button>
</body></html>

This rule changes the button’s background to orange when the user hovers the mouse over it. Other common pseudo-classes include :focus, :first-child, :nth-child(n), and :disabled.

Preview: Hover over the button to see the effect.

CSS Pseudo-Elements

Pseudo-elements style a specific part of an element (e.g. the first letter, the first line, or generated content before/after). They are written with two colons, e.g. ::before, ::first-letter.

Unlike pseudo-classes (which target a state), pseudo-elements target a subpart of the element’s content or create virtual content that you can style.

Common Pseudo-Elements

Selector Description
::beforeInserts content before the element (styled with CSS)
::afterInserts content after the element
::first-letterStyles the first letter of text
::first-lineStyles the first line of text
::markerStyles list markers (bullets, numbers)
::selectionStyles text when selected by the user

Example of Pseudo-Elements

p::first-letter {
  font-size: 30px;
  color: red;
}

This rule styles only the first letter of each paragraph: larger and red. The rest of the paragraph is unchanged.

Preview:

A paragraph with a styled first letter. The rest of the text stays normal.

Lecture Summary

Key Takeaways

  • CSS selectors target HTML elements so you can apply styles to them.
  • Descendant selectors (e.g. ol li) select elements inside other elements at any depth.
  • Child selectors (e.g. ol > li) select only direct children.
  • Attribute selectors (e.g. a[href="..."]) target elements by attribute values.
  • Pseudo-classes (e.g. :hover) style elements based on state.
  • Pseudo-elements (e.g. ::first-letter, ::before) style specific parts of elements.
